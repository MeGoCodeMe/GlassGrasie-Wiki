# Executive Summary

#strategie #kernboodschap

## De kans in één zin
Geen enkel Nederlands bedrijf transformeert industrieel restfloatglas naar premium designproducten — [[MAGNA-Glaskeramik]] bewijst dat de markt bestaat, en Glass⏧Grasie® is de eerste Nederlandse speler.

## Wat we doen
[[Van-Dijken-Glas]] genereert snijresten, afgekeurde prints en restplaten die normaal afgevoerd worden als afval. Glass⏧Grasie® geeft dit materiaal een tweede leven met behoud van alle optische kwaliteiten.

## Het model
Hybride B2C + B2B via vier productlagen:
1. [[Product-Sieraden]] — €35–145, start week 3
2. [[Product-Woonaccessoires]] — €65–650, start maand 2
3. [[Product-Kunstobjecten]] — €800–5.000, start maand 4
4. [[Product-B2B-Architectuur]] — €350–850/m², start maand 3
